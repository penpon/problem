🎯 配列の基本構造

配列は複数の要素をインデックス（番号）で管理するデータ構造です：

// 配列の作成 let fruits = ["りんご", "バナナ", "オレンジ"]; // インデックスでアクセス（0から開始） let first = fruits[0]; // "りんご" let second = fruits[1]; // "バナナ" let third = fruits[2]; // "オレンジ" // 配列の長さ let count = fruits.length; // 3

🔧 基本操作

  • 📋 配列表示: 配列の全内容とインデックスを確認
  • 🔍 要素アクセス: 指定したインデックスの要素を取得
  • ➕ 要素追加: push()で配列の末尾に要素を追加
  • ➖ 要素削除: pop()で配列の末尾から要素を削除
