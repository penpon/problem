🎯 ランダム選択の仕組み

配列からランダムに要素を選択する基本的なプロセス：

// ステップ1: 配列を準備 let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"]; // ステップ2: 0以上1未満のランダムな小数を生成 let randomFloat = Math.random(); // 例: 0.6789 // ステップ3: 配列の長さと掛け算 let scaled = randomFloat * fruits.length; // 例: 0.6789 × 4 = 2.7156 // ステップ4: 整数のインデックスを取得 let randomIndex = Math.floor(scaled); // 例: Math.floor(2.7156) = 2 // ステップ5: ランダムな要素を取得 let randomFruit = fruits[randomIndex]; // 例: fruits[2] = "オレンジ"

🔧 実装されている機能

  • 🍎 果物ランダム: 5種類の果物からランダム選択
  • 🎨 色ランダム: 6色からランダム選択（背景色も変更）
  • 🎵 音楽ランダム: 5つの音楽ジャンルからランダム選択
  • 🎯 全混合ランダム: 全配列（16要素）から1つをランダム選択
