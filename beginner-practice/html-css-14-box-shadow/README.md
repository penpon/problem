# 🎯 ステップ11: 影（box-shadow）の基本

## 📖 学習内容
要素に影をつけて**立体感**を演出する「**box-shadow**」プロパティを学習します。フラットなデザインに深みを与える重要な技術です。

## 🎯 学習目標
- `box-shadow`で影を作れる
- 影の濃さや距離を調整できる
- 内側の影（inset）を理解する

## 📝 新しく学ぶこと
1. **box-shadow**: 影を作るプロパティ
2. **影の4つの値**: 横・縦・ぼかし・色
3. **inset**: 内側に影を作る

## 🔍 詳細解説

### box-shadowの基本構文
```css
box-shadow: 横の距離 縦の距離 ぼかしの距離 色;
```

### 実際の例
```css
.basic-shadow {
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.2);
}
```
- `2px`: 右に2px移動
- `2px`: 下に2px移動  
- `5px`: 5pxぼかし
- `rgba(0, 0, 0, 0.2)`: 透明な黒（20%）

### 内側の影（inset）
```css
.inset-shadow {
    box-shadow: inset 0 2px 4px rgba(0, 0, 0, 0.2);
}
```
- `inset`をつけると内側に影
- 押し込まれた印象になる

### rgba色の指定
- `rgba(0, 0, 0, 0.1)`: うっすらとした影
- `rgba(0, 0, 0, 0.2)`: 標準的な影
- `rgba(0, 0, 0, 0.4)`: 濃い影

## ✨ 実際に試してみよう
1. 各影の違いを確認
2. ぼかしの値を変更してみる
3. 透明度を調整してみる
