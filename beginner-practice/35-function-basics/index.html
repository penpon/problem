🔧 関数の基本形

パラメータなし関数

function sayHello() { console.log("こんにちは！"); } sayHello(); // 関数を呼び出し

最もシンプルな関数の形

パラメータあり関数

function greet(name) { console.log("こんにちは、" + name + "さん"); } greet("田中"); // 引数を渡す

値を受け取って処理する関数

戻り値のある関数

function double(number) { return number * 2; // 結果を返す } let result = double(5); // result には 10 が入る
