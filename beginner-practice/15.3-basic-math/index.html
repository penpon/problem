🔢 JavaScriptの四則演算

+ 足し算（たしざん）

let result = 5 + 3; // 結果: 8

- 引き算（ひきざん）

let result = 10 - 4; // 結果: 6

* 掛け算（かけざん）

let result = 6 * 7; // 結果: 42

/ 割り算（わりざん）

let result = 15 / 3; // 結果: 5
