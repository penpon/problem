# 🧪 js-basic-15: フォーカス/ブラーで見た目を変える

## 🎯 学習目標
- `focus` と `blur` イベントで、入力中の視覚的フィードバックを与えられる。

## 📖 内容
- 入力がフォーカスされたら強調、外れたら元に戻す

## 📝 学習ポイント
- 1ステップ1概念: 新規は`focus/blur`のみ
- クラスの付け外し（`classList.add/remove`）は補助的に使用

## 🔍 詳細解説
- 入力フォームの使い勝手向上。ECの住所/カード入力での強調に有効
