🎯 複数ボタンの管理方法

複数のボタンを効率的に管理するために、以下の手順で実装します：

// 1. 各ボタンを個別に取得 let welcomeBtn = document.getElementById("welcome-btn"); let happyBtn = document.getElementById("happy-btn"); // 2. それぞれにイベントを設定 welcomeBtn.addEventListener("click", function() { // ようこそボタン用の処理 }); happyBtn.addEventListener("click", function() { // 幸せボタン用の処理 });

👀 実装されているコード例

// ようこそボタンの処理 document.getElementById("welcome-btn").addEventListener("click", function() { let display = document.getElementById("message-display"); display.textContent = "👋 ようこそ！JavaScriptの学習へ！"; display.style.background = "linear-gradient(135deg, #667eea, #764ba2)"; }); // 幸せボタンの処理 document.getElementById("happy-btn").addEventListener("click", function() { let display = document.getElementById("message-display"); display.textContent = "😊 今日は良い一日ですね！笑顔でいきましょう！"; display.style.background = "linear-gradient(135deg, #ffecd2, #fcb69f)"; });

🚀 すべてのボタンを試してみよう！

6つのボタンをすべてクリックして、それぞれ異なるメッセージと背景色の変化を確認してください。

F12（MacはCmd+Option+I）で開発者ツールを開き、Consoleタブでクリックの詳細ログも確認できます！
