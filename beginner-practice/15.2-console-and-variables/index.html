ステップ15.2 - プログラマーの秘密の道具
🔍 📋

📚 このステップで学ぶこと

  • コンソールでプログラムの動作を確認する方法
  • 変数でデータを保存する方法
  • ブラウザの開発者ツールの使い方

🚀 重要：開発者ツールを開いてください！

このページの学習効果を最大化するため、以下の手順で開発者ツールを開いてください：

  • Windows/Linux: F12キーを押す、または右クリック→「検証」
  • Mac: Cmd + Option + I、または右クリック→「検証」

開いたら「Console」タブをクリックしてください。

コンソールにメッセージが表示されているはずです！

🖥️ コンソールとは？

コンソールは、プログラマーが使う「デバッグツール」です。

プログラムの動作を確認したり、エラーを見つけたりするのに使います。

まるでプログラムとの「チャット画面」のようなものです！

📋 変数とは？

変数は、データを入れておく「箱」のようなものです。

let userName = "田中さん"; let userAge = 25;

この例では、userNameという箱に「田中さん」を、userAgeという箱に「25」を入れています。

👀 実際のコードを見てみよう

このページでは、以下のJavaScriptコードが実行されています：

// ようこそメッセージ let message = "ステップ15.2へようこそ！"; console.log(message); // 名前を変数に保存 let studentName = "未来のプログラマー"; console.log("こんにちは、" + studentName + "さん！"); // 学習の進捗 let stepNumber = 15.2; console.log("現在のステップ：" + stepNumber);

✅ 確認してみよう

開発者ツールのコンソールに、以下のメッセージが表示されていることを確認してください：

  1. 「ステップ15.2へようこそ！」
  2. 「こんにちは、未来のプログラマーさん！」
  3. 「現在のステップ：15.2」
