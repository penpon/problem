💡 setTimeout()の基本構造

setTimeout()は指定した時間後に1回だけ関数を実行します：

// 基本的な使い方 setTimeout(function() { console.log("3秒後に実行されました！"); }, 3000); // 3000ミリ秒 = 3秒後 // HTMLを変更する例 setTimeout(function() { document.getElementById("message").textContent = "タイマー完了！"; }, 2000); // 2秒後にメッセージを変更
