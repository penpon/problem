🔧 従来の関数 vs アロー関数

従来の関数

function greet(name) { return "こんにちは、" + name + "さん"; } let message = greet("田中");

アロー関数

const greet = (name) => { return "こんにちは、" + name + "さん"; }; let message = greet("田中");
