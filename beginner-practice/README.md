# 🌟 超初心者向けWeb開発学習サイト

## 📋 概要

このプロジェクトは、**完全にWeb開発が初めての方**のための段階的学習サイト集です。HTMLタグを初めて学ぶレベルから始まり、**26段階**で基本的なWebサイト制作から**プロフェッショナルレベルの動的Webシステム**まで身につけられます。

## 🎯 対象者

- **HTMLを初めて学ぶ方**
- **プログラミング経験がない方**
- **挫折せずに確実にスキルアップしたい方**

## 📚 学習の特徴

### ✅ **超小ステップ方式**
- 各ステップで学ぶことは**1つだけ**
- ファイル行数は10行から徐々に増加
- 前のステップを完全に理解してから次へ進む

### ✅ **挫折しない学習曲線**
- 急激な難易度上昇なし
- 段階的にスキルを積み上げ
- 達成感を感じながら学習できる

### ✅ **実践的な学習**
- 最終的に実用的なWebサイトを制作
- 各ステップで実際に動くものを作る
- 学習成果を目で確認できる

### ✅ **基礎から発展まで完全対応**
- **基礎編（1-20）**: HTML・CSS・JavaScript の基本をマスター
- **発展編（20.1-20.6）**: 実際のECサイトレベルの高度な機能を実装
- 初心者から中級者レベルまでの技術習得が可能

## 🚀 学習の始め方

### **1. 順番に進める**
必ず **01** から順番に進んでください。前のステップで学んだ内容が次のステップで使われます。

**📚 学習の流れ:**
- **基礎編（1-20）**: HTML・CSS・JavaScriptの基本を習得
- **発展編（20.1-20.6）**: プロフェッショナルレベルの動的システム構築
- **発展編は基礎編完了後に挑戦**してください（ステップ20完了が前提）

### **2. 実際に動かす**
各ステップの `index.html` をブラウザで開いて、実際に動作を確認してください。

### **3. コードを読む**
なぜそのコードが必要なのか、READMEの説明を読みながら理解を深めてください。

### **4. 改造してみる**
慣れてきたら色を変えたり、文字を変えたりして遊んでみてください。

## ✅ 各ステップの完了条件

各ステップで以下を確認してから次に進みましょう：

- [ ] **ブラウザで表示確認**：正しく表示されることを確認
- [ ] **コード理解**：なぜこのコードが必要かを理解
- [ ] **小さな改造**：色や文字を変更してみる
- [ ] **質問に答える**：「このタグは何をしているか？」など
