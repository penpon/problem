💡 パラメータと戻り値の基本

関数にデータを渡し、計算結果を受け取る方法：

// パラメータを受け取る関数 function add(a, b) { // a と b がパラメータ let result = a + b; // 計算処理 return result; // 戻り値として結果を返す } // 関数を呼び出して戻り値を取得 let sum = add(5, 3); // 5 と 3 を渡して、8 が返される console.log(sum); // 8 が出力される // 別の値でも使える let sum2 = add(10, 20); // 30 が返される
