💡 複数配列管理の基本

複数の配列を効率的に管理する方法：

// 1. 複数の配列を定義 let quotes = ["努力は必ず報われる", "継続は力なり"]; let colors = ["赤", "青", "緑"]; let foods = ["ピザ", "ハンバーガー", "寿司"]; // 2. ランダム選択関数を作成 function randomSelect(array) { let index = Math.floor(Math.random() * array.length); return array[index]; } // 3. 各配列から選択 let randomQuote = randomSelect(quotes); let randomColor = randomSelect(colors); let randomFood = randomSelect(foods);
