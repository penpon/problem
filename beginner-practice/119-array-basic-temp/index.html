💡 配列の基本構造

配列は複数のデータを1つの変数にまとめて保存する仕組みです：

// 1. 配列を作成 let fruits = ["りんご", "バナナ", "オレンジ"]; // 2. 配列の長さを確認 console.log(fruits.length); // 3 // 3. 特定の要素にアクセス（インデックスは0から始まる） console.log(fruits[0]); // "りんご" console.log(fruits[1]); // "バナナ" console.log(fruits[2]); // "オレンジ"
