💡 関数の基本構造

関数は「処理をまとめて名前を付けたもの」です：

// 1. 関数を定義（処理をまとめる） function sayHello() { console.log("こんにちは！"); console.log("関数が実行されました"); } // 2. 関数を呼び出し（実際に実行する） sayHello(); // "こんにちは！" と "関数が実行されました" が出力される // 何度でも呼び出せます sayHello(); sayHello();
