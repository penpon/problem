# 🧪 js-basic-14: リンクの既定動作を止める

## 🎯 学習目標
- クリック時に `event.preventDefault()` を使い、ページ遷移を防止できる。

## 📖 内容
- `<a>` のクリックに反応
- 既定動作を止め、メッセージと回数を表示

## 📝 学習ポイント
- `addEventListener('click', handler)` と `handler(event)`
- `event.preventDefault()` の意味

## 🔍 詳細解説
- ECで「#」リンクや疑似ボタンの誤遷移防止に必須
