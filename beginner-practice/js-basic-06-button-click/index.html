🎯 addEventListener とは？

addEventListenerは、HTML要素に「イベント」を設定するJavaScriptの機能です。

// ボタンを取得 let button = document.getElementById("my-button"); // クリック時の動作を設定 button.addEventListener("click", function() { alert("ボタンがクリックされました！"); });

この設定により、ボタンがクリックされた時に指定した処理が実行されます。
