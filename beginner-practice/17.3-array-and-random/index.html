🎯 配列とランダム選択の仕組み

配列からランダムに要素を選択する基本的な方法：

// 1. 配列を定義 let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"]; // 2. ランダムなインデックスを生成（0から配列長-1まで） let randomIndex = Math.floor(Math.random() * fruits.length); // 3. ランダムな要素を取得 let randomFruit = fruits[randomIndex]; // 結果: "りんご", "バナナ", "オレンジ", "ぶどう" のいずれかが選ばれる

🔧 実装されている機能

  • 💬 名言ランダム: やる気の出る名言をランダム表示
  • 🎨 色ランダム: カラフルな色をランダム選択（背景色も変更）
  • 🐾 動物ランダム: 可愛い動物たちをランダム表示
  • 🍕 食べ物ランダム: 美味しそうな食べ物をランダム選択
  • 🎯 趣味ランダム: 様々な趣味活動をランダム提案
  • 🔢 数字ランダム: 1-100の数字をランダム生成
  • 🌟 全混合ランダム: 全配列から1つをランダム選択
  • 🔄 配列シャッフル: 配列の順番をランダムに並び替え
