# 改訂版ヒント（03 画像の追加）

## 🧩 今回学ぶタグ
- `img`（主要属性：`src`, `alt`）

## 🔁 前回の復習
- 見出し `h1` で主題、`p` で本文を書く基本

## 📌 重要なポイント
- `img` は**空要素**（閉じタグなし）。`src` が場所、`alt` が内容説明
- `alt` は**アクセシビリティとSEO**の双方で重要
- レイアウト調整は**CSSで実施**（HTMLは意味付けに集中）

## 🧪 例題
課題とは別テーマの例：「季節の風景」。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8">
    <title>季節の風景</title>
  </head>
  <body>
    <h1>季節の風景</h1>
    <img src="../shared/images/sample-landscape.jpg" alt="夕暮れの湖と山並みの写真">
    <p>静かな湖に映る夕焼け空がとても印象的でした。</p>
  </body>
</html>
```

## ✨ 新しく追加された部分
- `<img src="…" alt="…">`：画像と代替テキストの指定

## 🔍 コードの説明
- `src`：画像ファイルへの**相対/絶対パス**
- `alt`：画像が表示できない/読めない環境での**代替説明**

## 📖 豆知識
- 遅延読み込み `loading="lazy"` は**パフォーマンス改善**に有効
- ベクター（SVG）とビットマップ（PNG/JPG）の**使い分け**
- サイズ調整はCSSの **`width`/`height`**, `object-fit` を利用

## ⚠️ 注意点
- 意味のある画像には**必ず`alt`**を入れる
- `../` は**1つ上の階層**を指す（相対パスの理解）
- 画像は**最適化**して読み込みを高速化

## 🛒 ECサイト制作で繋がるポイント
- 商品画像はUXの要。`alt`に**「商品名＋特徴」**を入れると検索性も向上
- `/images/products/` などの**ディレクトリ設計**が保守性に直結
