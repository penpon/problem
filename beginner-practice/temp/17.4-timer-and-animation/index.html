⏰ setTimeout と setInterval の違い

setTimeout（単発実行）

setTimeout(function() { console.log("3秒後に実行"); }, 3000);

指定時間後に1回だけ実行

setInterval（繰り返し実行）

setInterval(function() { console.log("1秒ごとに実行"); }, 1000);

指定間隔で繰り返し実行

🎬 実装されている機能

  • ▶️ 開始/再開: タイマーを開始または一時停止から再開
  • ⏸️ 一時停止: タイマーを一時停止（時間は保持）
  • 🔄 リセット: タイマーを00:00:00に戻す
  • ⏳ カウントダウン: 10秒からのカウントダウンタイマー
  • 🔔 定期実行: 2秒ごとにアニメーション要素を移動
  • 🎨 自動変化: 1秒ごとにテキスト・色・アニメーションを変更

🚀 すべてのタイマー機能を試してみよう！

6つのボタンをすべてクリックして、時間ベースの動的変化を体験してください。特に「自動変化」ボタンでは、複数の要素が同時にアニメーションすることを確認しましょう。
