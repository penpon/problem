⚡ 関数の種類と書き方

function宣言

function sayHello() { return "こんにちは！"; } let result = sayHello();

最も基本的な関数宣言方法

アロー関数（ES6）

const sayHello = () => { return "こんにちは！"; }; let result = sayHello();

より短く記述できる新しい記法

🔧 実装されている関数機能

  • 👋 基本関数: パラメータなし、戻り値なしの最もシンプルな関数
  • 📝 パラメータ付き: 入力値を受け取って処理する関数
  • 🔄 戻り値あり: 計算結果を返す関数
  • 🏹 アロー関数: ES6の新しい記法での関数定義
  • 🧮 複数パラメータ: 複数の値を受け取って計算する関数
  • 📊 BMI計算システム: 実用的な計算を行う関数の組み合わせ
